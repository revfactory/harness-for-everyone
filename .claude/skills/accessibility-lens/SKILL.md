# Accessibility Lens — 이해 접근성 3축 평가 렌즈

하네스 대중화 팀의 모든 산출물을 **이해 접근성** 관점에서 평가할 때 쓰는 공통 기준. concept-translator / learning-experience-designer / critical-reviewer / synthesis-lead가 각자의 산출물 완성 전에 자체 점검할 것. 본 하네스의 목적은 하네스 개념의 대중적 이해이며, 가격·제품 설계는 범위 밖이므로 평가 축에서 제외한다.

## 3축 정의

| 축 | 정의 | 대표 장벽 예 |
|----|------|-------------|
| 인식(Awareness) | 하네스의 존재와 의미를 알아차리는 첫 관문 | "하네스"라는 단어를 들어본 적 없음, 왜 알아야 하는지 모름, AI·에이전트·자동화와 어떻게 다른지 모름 |
| 학습(Learning) | 개념을 이해하기까지 필요한 인지 부담 | 전문 용어 밀도, 선행 지식 가정, 한 번에 도입되는 새 개념 수, 추상 정도 |
| 심리(Psychological) | 이해 과정에서 발생하는 거부·두려움·불신 | "어차피 나는 못 따라갈 것", "AI가 나를 대신 결정하는 건 무섭다", "전문가 영역이라 내가 알 필요 없다" |

## 산출물별 점검 질문

### 은유(concept-translator)
- 은유가 학습 부담 점수를 몇 점 낮추는가?
- 은유가 심리 불안을 키우지는 않는가? (예: "조종사", "대리인" 같은 권한 위임 은유는 심리 장벽↑)
- 은유가 문화권·세대에 따라 낯설지 않은가? (특정 직업·취미 전제 은유 주의)

### 학습 경로(learning-experience-designer)
- 각 단계에서 새 용어가 3개 이하로 도입되는가?
- 선행 개념 없이 진입 가능한 "0단계" 문장이 존재하는가?
- 단계별 도달 목표가 관찰 가능한 행동으로 서술되었는가?
- 이해 실패 시 심리적 자책을 유발하지 않는 설명 구조인가?

### 비판(critical-reviewer)
- 전체 제안이 "기술 친화적인 사람에게만" 통하는 설명은 아닌가?
- 장벽을 한 축만 해결하고 다른 축을 높이지는 않는가? (예: 인식은 쉬워졌지만 심리는 오히려 불안해짐)
- 오개념 카탈로그가 3축 중 어느 축에 영향을 주는지 태깅되었는가?

### 통합(synthesis-lead)
- Executive Summary가 3축 종합 점수를 포함하는가?
- 양안 병기된 항목이 어느 축에서 상충하는지 명시되었는가?

## 점수 카드

각 산출물을 3축 × 5점 척도로 자체 채점하고 산출물 말미에 기록한다:

```
Accessibility Score
- 인식 낮춤: ⬤⬤⬤⬤○ (4/5) — 은유·도입 훅이 존재와 의미를 명료히 전달
- 학습 낮춤: ⬤⬤⬤⬤⬤ (5/5) — 선행 지식 없이 0단계부터 진입 가능, 새 용어 밀도 낮음
- 심리 낮춤: ⬤⬤⬤○○ (3/5) — "내가 이해할 수 있다"는 감각 강화 설명 보강 필요
```

## 트레이드오프 원칙

- 한 축만 극단적으로 낮추고 다른 축이 올라가면 **순 이해 장벽은 증가**할 수 있다
- 3축이 모두 "평균 이상"인 설계가 대중적 이해에 이긴다
- 심리 축은 자주 간과됨 — 쉬운 은유와 짧은 학습 경로가 있어도 "나는 못 따라갈 것"이라는 신념이 있으면 이해가 시작되지 않는다

## 사용 시점

각 에이전트는 자기 산출물을 저장하기 전 이 스킬을 Read하고, 산출물 말미에 자체 3축 점수 카드를 첨부한다. synthesis-lead는 제안서 작성 시 3축 종합 점수를 Executive Summary에 포함한다.

## 범위 외 (명시적 제외)

다음 평가 축은 본 하네스의 목적(개념 이해 전달) 범위 밖이므로 이 렌즈에서 제외한다:
- 가격·지불·요금제 장벽 — 제품화·상용화 하네스가 아닌 '이해' 하네스이므로 대상 아님
- 제품 UX·설치·실행 장벽 — 터미널/CLI 의존성 제거 설계는 본 프로젝트 목적이 아님
- 기술 아키텍처·인프라 비용 — 구현/운영 차원의 접근성은 평가하지 않음

위 축에 관한 지적이 들어오면 critical-reviewer는 "범위 외"로 태깅하고 본문에서 분리한다.
