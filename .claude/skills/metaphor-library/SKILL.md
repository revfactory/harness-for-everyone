# Metaphor Library — 하네스 대중화 은유 설계 가이드

하네스의 추상 개념을 일반인이 직관적으로 이해하는 은유로 번역할 때 사용하는 패턴 라이브러리.

## 핵심 원칙

1. **익숙한 것에서 출발하라** — 사용자가 매일 경험하는 공간·활동·관계에서 은유를 찾는다
2. **감각을 자극하라** — 소리/질감/움직임이 연상되는 은유가 기억에 남는다
3. **실패 지점을 명시하라** — 모든 은유는 어느 지점에서 실제 개념과 어긋난다. 그 경계를 숨기지 말 것
4. **하나의 은유로 끝까지 끌고 가라** — 여러 은유를 혼합하면 오개념이 생긴다

## 핵심 개념 ↔ 후보 은유 매핑

| 하네스 개념 | 후보 은유 | 강점 | 실패 지점 |
|-----------|----------|------|----------|
| 하네스(Harness) | 주방 시스템 / 오케스트라 / 공장 라인 / 영화 제작 현장 / 스포츠팀 전술판 | 역할 분담과 협업이 직관적 | "실시간 무대"는 아니고 "연습 중"에 가까움 |
| 에이전트(Agent) | 셰프·조수·배우·선수·동료 | 사람처럼 생각하고 행동한다는 감각 전달 | 실제로는 기억이 휘발되고 동시에 수십 개 복제 가능 |
| 오케스트레이터 | 지휘자 / 총감독 / 헤드셰프 / 감독 | 전체를 아우르는 리더 역할 전달 | 지휘자는 즉흥 반응 불가, 오케스트레이터는 실시간 분기 가능 |
| 스킬(Skill) | 레시피 / 악보 / 매뉴얼 / 플레이북 | "참고해서 실행하는 문서"의 느낌 | 스킬은 상황마다 다르게 해석된다 (악보는 그렇지 않음) |
| 서브에이전트 | 인턴·외주·부엌 보조 / 세트 전문팀 | 메인이 시키고 결과만 받는 구조 | 인턴은 기억을 유지, 서브에이전트는 매번 새로 시작 |
| 에이전트 팀 | 회의실 / 작전실 / 공동 키친 / 스튜디오 | 상호 소통과 자율 조율 전달 | 사람 팀은 관계가 누적, AI 팀은 세션마다 리셋 |
| 컨텍스트 윈도우 | 데스크 위 공간 / 단기 기억 / 책상 작업대 | 한정된 공간 제약 전달 | 사람 단기 기억은 의미 압축이 자동, AI는 토큰 단위 |
| 프롬프트 | 주문서 / 대본 / 요청서 / 지시문 | "내가 시키는 것"의 느낌 | 주문서와 달리 말투와 맥락까지 결과에 반영됨 |
| 도구 호출(Tool) | 연장·장비 / 부엌 가전 | 행동의 수단 | 도구 사용이 "결정"까지 포함 |
| 컨텍스트/메모리 | 서랍장 / 파일 캐비닛 / 노트 | 꺼내 쓰는 정보 창고 | 검색·접근 방식이 사람과 다름 |
| 비용(토큰) | 전기 요금 / 택시 미터기 / 수도 사용량 | 쓸수록 누적 | 완성도와 비용이 비선형 관계 |
| 자율 실행 | 자동 주행 / 자율 청소기 / 자동 요리 | "내가 안 시켜도 한다"는 감각 | 예측 불가능성이 더 크다 |

## 3단 설명 레이어 템플릿

각 개념은 아래 3단 레이어로 제시한다:

### L1 — 한 줄 은유 (10초, 모든 사용자)
> "하네스는 AI 주방 브리게이드를 꾸리는 레시피 북이다."

### L2 — 3문장 풀이 (1분, 관심 있는 사용자)
> "AI에게 혼자 주문받게 하면 실수가 많습니다. 셰프·수세프·플레이터·디저트 담당처럼 역할을 나누고, 누가 언제 무엇을 만들지 적어두면 주방이 굴러갑니다. 하네스는 그 레시피 북이자 주방 구조도입니다."

### L3 — 실제 개념 매핑 (3~5분, 실행하려는 사용자)
> "에이전트 = 셰프. 스킬 = 레시피. 오케스트레이터 = 헤드셰프. 서브에이전트 = 단건 외주 요리사. 단, 이 주방은 매 서비스마다 기억을 잃기 때문에, 기록을 파일로 남겨야 합니다..."

## 은유 점검 질문

새 은유를 만들 때 다음을 체크한다:

- [ ] 내 어머니/이모/이웃 어르신이 이 은유를 듣고 다른 사람에게 전달할 수 있는가?
- [ ] 은유의 실패 지점이 3개 이상 있지는 않은가? (너무 많으면 은유가 잘못됨)
- [ ] 은유가 특정 직업·성별·계층에만 익숙하지 않은가? (주방/공장/운동/예술 중 다양성)
- [ ] 3단 레이어가 모두 같은 은유로 끝까지 이어지는가?
- [ ] 은유의 감각(시각·청각·운동감) 중 최소 하나가 선명한가?

## 문화권별 고려

| 문화권 | 강한 은유 후보 | 주의 |
|--------|-------------|------|
| 한국 | 주방 브리게이드, 아이돌 기획사, 드라마 제작 현장 | 계급적 뉘앙스 조심 |
| 북미 | 스포츠 팀 (미식축구 플레이북), 재즈 밴드, 파일럿 체크리스트 | 개인주의 톤 |
| 일본 | 장인 공방, 라멘 주방, 스튜디오 지브리 제작팀 | 위계·장인 정신 |
| 동남아 | 노점 주방, 가족 식당, 타운홀 | 공동체 톤 |

## 참조 시점

concept-translator가 새 은유를 제안할 때 이 라이브러리를 먼저 확인하고, 기존 후보를 활용하거나 새 후보를 추가한다. 새 은유 추가 시 위 표에 강점·실패 지점을 함께 기록한다.
