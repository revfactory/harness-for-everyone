## 1. 스타일 바이블

### 1.1 배경

- **기본 배경:** 크림색 종이 질감 (`#FFF8F0`)
- **종이 질감:** 미세한 섬유 결이 보이는 수제 종이 느낌. 완전한 흰색(#FFFFFF) 사용 금지
- **가장자리:** 종이 모서리에 미세한 그림자 또는 약간 너덜너덜한 에지 허용 (핸드메이드 감성)
- **패널 구분선:** 얇은 연필선 (0.5~1pt), 완벽한 직선 아닌 약간 떨리는 손그림 선

### 1.2 질감

- **주 질감:** 색연필 (colored pencil) — 연필 결이 살짝 보이는 빗금 채색
- **보조 질감:** 수성 마카 (water-based marker) — 넓은 면적 채색 시 사용. 마카 특유의 얼룩·겹침 흔적 허용
- **외곽선:** 연필 또는 가는 펜 (0.3~0.8pt) — 기계적으로 완벽한 선 금지
- **그림자:** 빗금(hatching) 또는 색연필 덧칠로 표현. 드롭섀도우·글로우 효과 금지
- **하이라이트:** 종이 배경색 노출(칠하지 않은 영역)로 처리

### 1.3 색상

4장 색상 팔레트 참조. 모든 색상은 파스텔 톤. 채도 40~60%, 명도 70~85% 범위 내.

### 1.4 타이포그래피

- **제목/헤드라인:** 손글씨(hand-lettered) 스타일. 굵고 둥근 글씨체. 약간의 기울임·크기 변화 허용
- **본문/설명:** 가독성 좋은 손글씨 또는 손글씨풍 폰트. 줄간격 넉넉하게
- **라벨/캡션:** 작은 손글씨. 화살표·밑줄과 함께 사용
- **말풍선 텍스트:** 둥근 손글씨. 말풍선 테두리도 손그림
- **숫자/강조:** 마카로 동그라미 또는 밑줄 처리
- **폰트 참고 (프롬프트용):** hand-lettered, handwritten, casual lettering

### 1.5 아이콘 스타일

- **형태:** 단순한 손그림 doodle 아이콘. 선 두께 일정하지 않아도 됨
- **크기:** 캐릭터 머리 크기의 1/3~1/2 이내
- **색상:** 주색 팔레트에서 단색 또는 2색 조합
- **용도별 아이콘:**
  - 하네스 = 펼쳐진 노트북/매뉴얼 아이콘
  - 에이전트 = 셰프 모자 아이콘
  - 스킬 = 레시피 카드 아이콘
  - 오케스트레이터 = 큰 국자 또는 벨 아이콘
  - 도구(Tool) = 오븐/믹서/칼 미니 아이콘
  - MCP = USB-C 포트 미니 아이콘
  - 메모리 = 접착 메모지(포스트잇) 아이콘
  - 컨텍스트 윈도우 = 조리대 위 공간(제한된 테이블) 아이콘

## 2. 캐릭터 디자인 시트

### 공통 규칙

- **비율:** 머리:몸 = 약 1:2.5 (귀여운 데포르메, SD가 아닌 세미리얼 doodle)
- **외곽선:** 연필선, 약간 떨리는 손그림. 완벽한 곡선 금지
- **채색:** 색연필 결이 살짝 보이는 파스텔 톤. 피부색은 다양성 반영
- **표정:** 기본 4종 (미소/놀람/생각/설명) + 캐릭터별 고유 1종
- **소품 일관성:** 각 캐릭터의 소품(모자 색, 앞치마 패턴 등)은 전 에피소드 동일 유지

**기본 비율:** 4:3 (가로:세로)

**패널 구성 패턴:**

```
패턴 A: 2단 균등 (기본)        패턴 B: 1대+2소
+----------+----------+       +----------+----+----+
|          |          |       |          |    |    |
|  패널 1  |  패널 2  |       |  패널 1  | 2  | 3  |
|          |          |       |  (큰장면) |    |    |
+----------+----------+       +----------+----+----+

패턴 C: 풀와이드 + 2소         패턴 D: 풀 페이지 (클라이맥스)
+---------------------+       +---------------------+
|      패널 1 (와이드) |       |                     |
+----------+----------+       |     단일 패널        |
|  패널 2  |  패널 3  |       |     (줌아웃 등)      |
+----------+----------+       |                     |
                              +---------------------+
```

**만화 페이지 규칙:**
- 페이지당 패널 수: 2~4개 (6개 초과 금지)
- 패널 간격: 패널 너비의 3~5%
- 패널 테두리: 손그림 연필선 (0.5~1pt), 약간 불규칙
- 읽기 방향: 왼쪽 -> 오른쪽, 위 -> 아래 (Z자 패턴)
- 말풍선: 패널 상단 1/3에 배치 권장, 캐릭터 머리 근처
- 여백: 페이지 가장자리에 최소 5% 여백

### 5.2 슬라이드 레이아웃

**기본 비율:** 16:9

**구도 패턴:**

```
패턴 A: 좌 일러스트 + 우 텍스트 (기본)
+----------+-------------------+
|          |  제목             |
| 일러스트  |  - 핵심 포인트 1   |
|  (40%)   |  - 핵심 포인트 2   |
|          |  - 핵심 포인트 3   |
+----------+-------------------+

패턴 B: 중앙 일러스트 + 하단 캡션
+---------------------+
|                     |
|    중앙 일러스트      |
|     (60% 높이)      |
+---------------------+
|  캡션 텍스트 (40%)   |
+---------------------+

패턴 C: 전면 일러스트 + 오버레이 텍스트 (표지/구분용)
+---------------------+
|  ///일러스트전면///   |
|    [텍스트 박스]     |
|    제목 + 부제      |
|  ///일러스트전면///   |
+---------------------+

패턴 D: 3등분 비교 (개념 매핑용)
+------+------+------+
|      |      |      |
| 개념1 | 개념2 | 개념3 |
| +그림 | +그림 | +그림 |
+------+------+------+
```

**슬라이드 규칙:**
- 슬라이드당 핵심 메시지 1개만
- 텍스트 최대 40단어 (제목 제외)
- 일러스트 영역 최소 40%
- 글머리 기호 최대 3개
- 전환 슬라이드(개념 간): 패턴 C 사용
- 데이터/매핑 슬라이드: 패턴 D 사용
- 폰트 크기: 제목 36pt 이상, 본문 20pt 이상 (손글씨풍)

### 5.3 반응형 고려 (모바일 카드뉴스)

- 세로형 비율 3:4 또는 9:16 변형 준비
- 텍스트 크기 20% 증대
- 패널당 캐릭터 최대 3명 (작은 화면 대응)
- 말풍선 텍스트 15자 이내 권장

- [ ] 크림색 종이 배경이 보이는가?

- [ ] 외곽선이 손그림(약간 떨림)인가?
- [ ] 캐릭터 비율이 1:2.5 근처인가?
- [ ] 광택/글로우/드롭섀도우가 없는가?
- [ ] 전체 분위기가 "따뜻하고 친근한가"?
